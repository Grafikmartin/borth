SoundPulse

Die Radio-App entstand im Rahmen des Frontend-Moduls als Teamprojekt mit zwei Mitstudenten. Design und Funktionalität wurden in Figma konzipiert, was eine strukturierte und professionelle Entwicklung ermöglichte. Technisch basiert die App auf React (Version 19) in Kombination mit Vite. Für Animationen kam Framer Motion zum Einsatz, die Internationalisierung erfolgte mit React i18next. Insgesamt wurden sieben Sprachen implementiert, verwaltet über einen zentralen TranslateContext. Zu den Hauptfeatures gehören ein responsives Design, intuitive Navigation, eine Rolling Gallery, eine Suchfunktion mit Filterbuttons sowie ein voll ausgestatteter Audioplayer mit Lautstärkeregelung. Die Benutzerverwaltung umfasst Registrierung, Login, Passwort-Vergessen-Funktion und Benutzereinstellungen – verwaltet über einen UserContext. Die Architektur ist komponentenbasiert, mit Context API für State Management und Error Boundary für Stabilität. Für das Styling werden separate Stylesheets pro Komponente verwendet. Tools wie ESLint, EmailJS, React Icons und die Radio Browser API ergänzen den Entwicklungsstack. Besondere Merkmale sind der Theme-Switch (ThemeContext), eine optimierte Bildvorladung sowie die modulare Struktur für einfache Wartung. Insgesamt vereint das Projekt moderne Webtechnologien mit nutzerfreundlichem Design und zeigt die Fähigkeit zur Teamarbeit und konzeptionellen Umsetzung.
