ZasterZen

ZasterZen ist eine moderne Web-App zur persönlichen Finanzverwaltung, entwickelt mit React und Material-UI. Sie bietet eine intuitive, responsive Benutzeroberfläche zur effektiven Verwaltung und Visualisierung von Finanzen. Hauptfunktionen sind ein Dashboard zur Finanzübersicht, Budgetverwaltung, Transaktionsverfolgung, Sparziele mit Fortschrittsanzeige, interaktive Berichte mit Chart.js, ein anpassbarer Dunkelmodus sowie Benutzerverwaltung mit Registrierung und Gastzugang. Die App nutzt Reacts Context API für das State Management und speichert Daten lokal via LocalStorage, wodurch keine externe Datenbank benötigt wird. Entwickelt mit Vite für schnelle Builds, setzt ZasterZen auf komponentenbasierte Architektur, reaktives UI-Design und die Integration von Drittanbieter-Bibliotheken zur Datenvisualisierung – mit Fokus auf Benutzerfreundlichkeit, Performance und Wartbarkeit.
